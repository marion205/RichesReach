# RichesReach Web/PWA

Progressive Web App version of RichesReach with Constellation Orb rendering for web browsers.

## Features

- ✅ **3D Constellation Orb** - Three.js-based interactive orb
- ✅ **PWA Support** - Installable, offline-capable
- ✅ **Responsive Design** - Works on desktop, tablet, mobile
- ✅ **Gesture Support** - Tap, swipe, pinch (touch + mouse)
- ✅ **SEO Optimized** - Server-side rendering ready

## Setup

```bash
npm install
npm run dev
```

## Build for Production

```bash
npm run build
npm run serve
```

## PWA Features

- Installable on desktop and mobile
- Offline functionality via service worker
- Fast loading with caching
- Share target support
