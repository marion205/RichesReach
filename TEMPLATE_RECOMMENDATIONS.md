## ðŸŽ¨ **DESIGN HACKS (Make It Look Premium Fast)**

### **Color Swaps (2 min)**
- Find template with good layout
- Use Canva's "Recolor" feature
- Swap to your brand colors
- Instant premium look

### **Font Upgrades (3 min)**
- Replace default fonts with:
  - **Headline:** Montserrat Bold or Poppins Bold
  - **Body:** Inter or Open Sans
- Makes huge difference

### **Logo Placement (2 min)**
- Top left or center
- Make it prominent but not overwhelming
- Add subtle background shape if needed

### **Number Emphasis (5 min)**
- Make numbers HUGE (72pt+)
- Use accent color
- Add subtle background box
- Instant "wow" factor

### **QR Code (3 min)**
- Use Canva's QR code generator
- Make it large (2" x 2" minimum)
- Add label: "Scan for demo" or "Download app"
- Place bottom right

## ðŸ–¨ï¸ **PRINTING TIPS**

### **File Setup:**
- **Size:** 8.5" x 11" (standard) or 8.5" x 8.5" (square)
- **Resolution:** 300 DPI minimum
- **Format:** PDF (print-ready)
- **Color:** CMYK (for professional printing) or RGB (for digital)

### **Paper Options:**
- **Standard:** 100lb cardstock (good quality)
- **Premium:** 110-120lb cardstock (feels expensive)
- **Finish:** Matte or gloss (matte = more premium feel)

### **Print Quantity:**
- **Test run:** 10 copies first
- **Full run:** 200 copies
- **Backup:** Keep digital PDF ready

### **Where to Print:**
- **Fast:** FedEx Office, Staples (same day)
- **Quality:** Local print shop (better paper options)
- **Online:** Vistaprint, Printful (if you have time)
