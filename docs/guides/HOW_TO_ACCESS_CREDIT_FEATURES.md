## üîß **Troubleshooting**

### If you don't see the credit button:
1. Make sure you're on the **Portfolio Screen**
2. Look in the **top-right corner** of the header
3. The button should be next to the Family Sharing (users) icon

### If the Credit Quest Screen is empty:
1. The app uses mock data by default
2. To see real data, you'll need to:
   - Link your bank accounts (via Yodlee)
   - Add credit card information
   - The app will sync transaction data automatically

### If notifications aren't working:
1. Check device notification settings
2. Make sure the app has notification permissions
3. Check the Credit Notification preferences in the app
